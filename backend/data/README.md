# Raw Data Files

The raw dataset files used by the backend are intentionally not committed to this repository because they exceed GitHub's file size limits.

Please download the required data files from the Canvas assignment page:
https://alueducation.instructure.com/courses/2779/assignments/39695

Required files:
- `yellow_tripdata.parquet`
- `taxi_zone_lookup.csv`
- `taxi_zones/` (shapefile bundle including `taxi_zones.shp`)

Once downloaded, place them under this folder:

```text
backend/data/
```
